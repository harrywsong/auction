# Sound Effects

Place your custom audio files in this folder. Supported formats: `.mp3`, `.ogg`, `.wav`

## Expected files

| Filename | Plays when |
|---|---|
| `bid.mp3` | A captain places a bid |
| `sold.mp3` | Timer expires and player is sold to highest bidder |
| `no_sale.mp3` | Timer expires with no bids — player goes to reserve |
| `timer_low.mp3` | Timer drops below 5 seconds |
| `auction_start.mp3` | Admin starts the regular or reserve auction |
| `auto_assign.mp3` | A player is auto-assigned (last of their tier) |

## Tips
- Keep files short (0.5–2 seconds) for bid/sold sounds
- Use `.mp3` for best browser compatibility
- Free sound libraries: [freesound.org](https://freesound.org), [zapsplat.com](https://www.zapsplat.com)
